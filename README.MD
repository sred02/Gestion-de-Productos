#  Sistema de Gestión de Productos
 
Sistema web para la administración de productos, que permite crear, visualizar, editar y eliminar registros desde una interfaz sencilla e intuitiva.
 
---
 
##  Funcionalidades
 
- 📋 **Ver productos** — listado completo de todos los productos registrados
- ➕ **Agregar producto** — formulario para registrar nuevos productos
- ✏️ **Editar producto** — modificar la información de un producto existente
- 🗑️ **Eliminar producto** — eliminar un producto del sistema
- 🏠 **Menú principal** — navegación central entre todas las secciones
---
 
##  Tecnologías utilizadas
 
| Capa | Tecnología |
|------|-----------|
| Backend | Python |
| Frontend | HTML, CSS |
| Base de datos | Neon (PostgreSQL serverless) |
